# StageOps

StageOps é uma plataforma em desenvolvimento para gestão de locação de equipamentos de eventos.

O projeto tem como objetivo ajudar empresas de som, iluminação, LED, palco, backline e estruturas a organizarem:

- Orçamentos
- Estoque de equipamentos
- Reservas por data
- Checklists de montagem
- Logística de carregamento e retorno
